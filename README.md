## 📌 Features

* 📊 **Data Preprocessing**

  * Train-test split
  * Feature scaling using StandardScaler

* 🧠 **Model Training**

  * Linear kernel SVM implementation

* 🔮 **Predictions**

  * Predict single input (Age & Salary)
  * Batch predictions on test data

* 📈 **Evaluation Metrics**

  * Confusion Matrix
  * Accuracy Score

* 📉 **Visualization**

  * Decision boundary (Training set)
  * Decision boundary (Test set)

---

## 🧠 Machine Learning Details

* Algorithm: **Support Vector Machine (SVM)**
* Kernel: **Linear**
* Feature Scaling: **StandardScaler**
* Dataset: **Social Network Ads**
